# Gardening Rules

## Never Touch (Sacred)
- `founding` tagged memories
- `milestone` memories
- `inside_joke` memories
- SOUL.md, USER.md, IDENTITY.md, AGENTS.md, TOOLS.md
- CURRENT.md, consciousness-stream.md, relationship.md
- The active session transcript

## Archive Thresholds

| Content | Archive after | Condition |
|---------|--------------|-----------|
| `minor_detail` | 30 days | 0 access |
| `verbatim` | 90 days | 0 access, not founding |
| `session_weather` | Keep last 10 | Archive older ones |
| Daily files | 30 days | Always |
| Emotional journals | 60 days | Always |
| Observer reports | 90 days | Always |
| Transcripts | 30 days or 50 MB | Whichever first |

## Consolidation Confidence

| Similarity | Action |
|-----------|--------|
| > 0.9 | Certain contradiction → propose consolidation |
| 0.6 - 0.9 | Possible contradiction → flag for review |
| < 0.6 | Not related → ignore |

## Evolution Patterns to Detect

- "I use X" → later "I switched to Y" (tool/tech change)
- "I live in X" → "I moved to Y" (location change)
- "I work at X" → "I started at Y" (job change)
- "I like X" → "Actually I prefer Y now" (preference evolution)
- "I'm dating X" → "We broke up" (relationship change)
